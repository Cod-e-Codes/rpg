## Controls

- **WASD / Arrow Keys** - Move
- **E** - Interact
- **1-5** - Cast spell (slot)
- **B** - Spellbook
- **I** - Inventory
- **ESC / P** - Pause
- **F3** - Debug info
- **F12** - Dev mode

## Dev Mode

Press F12 to toggle dev mode:
- Jump to any level
- Give yourself items/spells
- Speed multiplier
- View debug info

## Save Files

Save files are stored in your LOVE save directory:
- Windows: `%APPDATA%\LOVE\RPG\`
- Files: `save_slot_1.sav`, etc.
